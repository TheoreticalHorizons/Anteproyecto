Análisis experimental del péndulo doble

y su relación con la formulación lagrangiana

John Mauricio Mesa Echeverri
Aleph Sebastián Enciso Cruz
Universidad de Antioquia

Objetivo general

Mostrar que el péndulo doble permite observar comportamientos caóticos y evidenciar sensibilidad a condiciones iniciales, introduciendo conceptos de sistemas no lineales.

Objetivos específicos

  • Construir un péndulo doble funcional
  • Registrar el movimiento angular
  • Analizar los datos con Tracker
  • Formular y simular el modelo lagrangiano
  • Resolver las ecuaciones con Python
  • Comparar resultados teóricos y experimentales

Introducción (1/2)

  • Mecánica clásica: Newton → Lagrange
  • Coordenadas generalizadas y energías
  • Modelado con restricciones complejas

Introducción (2/2)

  • Péndulo doble: sistema determinista y caótico
  • Sensibilidad a condiciones iniciales
  • Ejemplo ideal de sistemas no lineales

Modelo teórico

Coordenadas: θ1, θ2

Lagrangiano: L = T - V

Ecuaciones de Euler-Lagrange:

∂L/∂qi - d/dt (∂L/∂̇qi) = 0

Planteamiento del problema

  • ¿Un montaje casero puede capturar el caos?
  • ¿Es comparable con la teoría?
  • ¿Qué limitaciones experimentales existen?

Hipótesis

El montaje permite visualizar caos cualitativamente, aunque con diferencias por fricción y errores.

Metodología

  4. Modelo lagrangiano
  5. Simulación numérica en Python
  6. Comparación teoría-experimento
